# แผนการทำงาน สาย 3 — เปิดออนไลน์ให้ทุกคนเห็นข้อมูลชุดเดียวกัน

ไฟล์ในชุดนี้ทำให้เว็บมี **ฐานข้อมูลกลาง** ทุกคนกรอกแล้วเห็นข้อมูลเดียวกันทุกเครื่อง/มือถือ
(เก็บข้อมูลบน **Turso** = SQLite บนคลาวด์ ฟรี + รันเว็บบน **Render** ฟรี)

ไฟล์ที่ต้องอัปขึ้น repo/Render:
- `index.html` · `support.js` · `data/seed.js` · `header.jpg`  (ตัวเว็บ)
- `server.js` · `package.json`  (เซิร์ฟเวอร์ + ฐานข้อมูล)

---

## ขั้นที่ 1 — สมัคร Turso แล้วสร้างฐานข้อมูล (ฟรี)

1. เข้า https://turso.tech → Sign up (ล็อกอินด้วย GitHub ได้)
2. สร้าง Database ใหม่ (กด **Create Database**) ตั้งชื่อ เช่น `plan-sai3` เลือก region ใกล้ไทย (เช่น Singapore)
3. เปิดฐานข้อมูลที่สร้าง → หน้า **Connect / Get URL** จะได้:
   - **URL** หน้าตาแบบ `libsql://plan-sai3-xxxx.turso.io`
   - **Auth Token** (กด **Create Token**) — เป็นข้อความยาวๆ
   เก็บค่า 2 อันนี้ไว้ ใช้ในขั้นที่ 3

> ตารางในฐานข้อมูลไม่ต้องสร้างเอง เซิร์ฟเวอร์สร้างให้อัตโนมัติตอนรันครั้งแรก

---

## ขั้นที่ 2 — อัปไฟล์ทั้งหมดขึ้น GitHub

อัป 6 ไฟล์ (รวมโฟลเดอร์ `data/`) ขึ้น repo ของคุณ
**อย่าอัป `node_modules/`**

---

## ขั้นที่ 3 — สร้างบริการบน Render

1. Render → **New → Web Service** (ไม่ใช่ Static Site)
2. เลือก repo ของคุณ
3. ตั้งค่า:
   - **Build Command:** `npm install`  (หรือ `yarn`)
   - **Start Command:** `node server.js`  (หรือ `yarn start`)
   - **Root Directory:** ถ้าไฟล์อยู่ในโฟลเดอร์ย่อย (เช่น `src`) ให้ใส่ชื่อโฟลเดอร์นั้น
4. เลื่อนไปที่ **Environment** → **Add Environment Variable** เพิ่ม 2 ตัว:
   - `TURSO_URL` = ค่า URL จากขั้นที่ 1
   - `TURSO_AUTH_TOKEN` = ค่า token จากขั้นที่ 1
5. กด **Create Web Service** → รอ build เสร็จ จะได้ลิงก์ เช่น `https://plan-sai3.onrender.com`

เปิดลิงก์ — มุมบนจะขึ้น **🟢 ออนไลน์ (ฐานข้อมูลกลาง)** แปลว่าต่อฐานข้อมูลกลางสำเร็จ

---

## ใช้งาน

- ใครกรอกข้อมูล เว็บจะบันทึกขึ้นฐานข้อมูลกลางอัตโนมัติ
- เครื่องอื่นจะเห็นข้อมูลอัปเดตเองภายในไม่กี่วินาที (รีเฟรชอัตโนมัติทุก 4 วินาที)
- ถ้าขึ้น **⚪ ออฟไลน์** = ต่อฐานข้อมูลไม่ได้ (ข้อมูลจะเก็บในเครื่องนั้นไว้ก่อน) — เช็คค่า TURSO_URL / TURSO_AUTH_TOKEN

> หมายเหตุ: ระบบใช้รูปแบบ "บันทึกทั้งก้อน" เหมาะกับทีมที่ไม่ได้กรอกพร้อมกันเป๊ะๆ หลายคน
> ถ้าต้องการกันชนกันแบบรายช่อง (หลายคนกรอกพร้อมกันจำนวนมาก) แจ้งได้ จะปรับให้
